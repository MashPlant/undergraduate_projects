Sample behavioral waveforms for design file piece_rom.vhd

The following waveforms show the behavior of altsyncram megafunction for the chosen set of parameters in design piece_rom.vhd. For the purpose of this simulation, the contents of the memory at the start of the sample waveforms is assumed to be ( 1F0, 1F1, 1F2, 1F3, ...). The design piece_rom.vhd has one read port. The read port has 32768 words of 9 bits each. The output of the read port is registered by clock.

Fig. 1 : Wave showing read operation.

The above waveform shows the behavior of the design under normal read conditions. The read happens at the rising edge of the enabled clock cycle. The output from the RAM is undefined until after the first rising edge of the read clock. The clock enable on the read side input registers are disabled. The clock enable on the output registers are disabled.
